2. Click: This Firefox
3. Click: Load Temporary Add-on
4. Select: extensions/firefox/manifest.json
```

### Microsoft Edge

```
1. Open: edge://extensions/
2. Enable: Developer mode ‚úÖ
3. Click: Load unpacked
4. Select: extensions/edge/ folder
```

### Safari

```
