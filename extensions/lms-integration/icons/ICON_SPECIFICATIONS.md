# Professor GENIE Extension Icons - Specifications

## Required Icons for Chrome Web Store

### Extension Icons (Required by manifest.json)

- **icon16.png** (16x16px) - Used in extensions management page
- **icon48.png** (48x48px) - Used in extensions management page and Chrome Web Store
- **icon128.png** (128x128px) - Used during installation and in Chrome Web Store

### Chrome Web Store Assets

- **store-icon.png** (128x128px) - Main store listing icon
- **screenshot1.png** (1280x800px) - Primary product screenshot
- **screenshot2.png** (1280x800px) - Secondary feature screenshot
- **promotional-tile.png** (440x280px) - Store promotional tile

## Design Guidelines

### Brand Colors (Updated Based on Genie Logo)

- **Primary Cyan**: #00b4d8 (Genie lamp cyan/blue)
- **Secondary Blue**: #0077b6 (Darker blue accent)  
- **Accent Gold**: #ffd60a (Genie lamp magical accent)
- **Background**: #f8f9fa (light gray)
- **Text**: #1f2937 (dark gray)

### Icon Design Requirements (Genie Lamp Theme)

1. **Professor GENIE Logo Elements**:
   - **Primary**: Stylized genie lamp (magic/wisdom symbol)
   - **Academic Touch**: Small graduation cap on lamp or integrated design
   - **Typography**: ":genie:" style text treatment or "PROFESSOR GENIE"
   - **Magic Elements**: Subtle sparkles, wisps, or magical effects

2. **Visual Style**:
   - Modern, flat design with magical gradient effects
   - Professional yet approachable appearance
   - Genie lamp as central recognizable element
   - Clear visibility at small sizes (16px)
   - Consistent branding across all sizes

3. **Design Hierarchy by Size**:
   - **16px**: Simple genie lamp silhouette in cyan
   - **48px**: Genie lamp with subtle magical effects and academic touch
   - **128px**: Full detailed lamp with sparkles, graduation elements, and "GENIE" text
   - Main element: Stylized graduation cap
   - Secondary element: AI/tech accent (small gear, circuit, or spark)
   - Color scheme: Blue gradient with purple accents
   - Typography: Clean sans-serif for any text elements
